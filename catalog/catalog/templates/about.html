About This Site

What is this?

LendingLibrary is a site developed using Flask as part of my work toward a Nanodegree in Full Stack Web Development offered by Udacity. The site enables a team or other group of people to share books they own for others in the group to borrow. The site offers pretty comprehensive functionality, including:

  • Ability to create an account and sign in using Google+ or Facebook.
  • Ability to see detailed info about each book in the Library including whether it is checked out (and when it's due back).
  • Ability to add books to the database.
  • Ability for a lender to edit or remove books s/he added to the Library.
  • Ability to check out and return books.
  • Ability to write reviews of books in the database.
  • Ability to mark books as "favorites".
  • Ability to search for books in the library by author, title, or synopsis.
  • Ability to browse books by genre (category).
  • Integration with the Google Books API to pull in cover art and other metadata.
  • Complete set of unit tests to ensure a working application.

This is a working app and is open source, so feel free to use!


Shout Outs

This site was developed using the following third party add-ons in addition to Flask and some Flask extensions:

  • RateYo!, a jQuery plugin for handling 5-star ratings using SVG.

Bootstrap, FontAwesome and jQuery were also used, but doesn't that kinda go without saying these days?
